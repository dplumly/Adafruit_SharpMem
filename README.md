Adafruit_SharpMem
=================

Adafruit Sharp Memory Display Library adapted for Spark

Pin setup for Spark and 96x96 Sharp Memory Display


Spark         | Sharp
------------- | -------------
3.3           | 3.3
GND           | GND
A3            | CLK
A3            | DI
A1            | CS
